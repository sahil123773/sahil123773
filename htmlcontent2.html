HTML Form

element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.. The HTML element is the most used form element. An input element can be displayed in many ways, depending on the type attribute.

HTML Form Attributes

action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.
